# Nature Background Images

Place your nature background images in this folder (`static/img/`).

## Required Images:

1. **nature-bg-main.jpg** - Main page background (full screen, fixed) and Hero section
   - Recommended: Forest, garden, or nature landscape
   - Size: 1920x1080 or larger
   - Format: JPG, PNG, or WebP
   - Used for: Body background and hero sections

2. **nature-bg-header.jpg** - Header/navbar background
   - Recommended: Subtle nature scene (leaves, plants)
   - Size: 1920x200 or larger
   - Format: JPG, PNG, or WebP
   - Used for: Top navigation bar

## Notes:

- All images will have semi-transparent overlays for text readability
- Images should be high quality but optimized for web
- Cards and other elements use the original light green color scheme
- Supported formats: JPG, PNG, WebP

## To Change Images:

Simply replace the image files with your own, keeping the same filenames:
- `nature-bg-main.jpg`
- `nature-bg-header.jpg`

Or update the CSS variables in `static/css/style.css` (lines 82-83) to use different filenames.
